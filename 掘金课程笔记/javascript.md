# 高阶函数
once ：为了能够执行一次的需求覆盖不同的事件处理，我们可以将这个需求剥离出来
这个过程称为过程抽象


1. 以函数为参数
2. 以函数为返回值
3. 常用于作为函数装饰器

模板：function HOF0(fn){
    return function(...args){
        return fn.apply(this,args)
    }
}
1. once
2. Throttle（适用于高频出发事件scroll mousemove）
3. Debounce (防抖，一直在输入的时候不给反应知道停止的时候)
